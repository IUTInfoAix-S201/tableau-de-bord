Comment lire ce tableau

  • Voyant qualité de revue (par contributeur) : vraies revues (commentaires, changements demandés), engagement léger, approbations à vide (tampon), pas encore de revue.
  • MVP ✅ = toutes les fonctionnalités MUST (dont l'écran pivot Passage) terminées.
  • Bus factor : part des commits portée par une seule personne ; élevé = travail déséquilibré.
  • Badges (tableau par étudiant) : 🏗️ le plus de commits · 🔍 le plus de revues · 🚀 le plus de PR mergées · 💬 le plus de commentaires de revue · 🐝 actif partout (code + PR + revues) · 🤝 relit autant qu'il est relu · 🧐 vraies revues exigeantes · 🦆 approbations à vide · 👻 passager clandestin. Survolez un badge pour le détail.

⚠️ Les voyants et indicateurs collaboratifs sont des repères heuristiques, indicatifs et non des notes. Les chiffres bruts sont toujours affichés pour que chacun puisse les interpréter.
